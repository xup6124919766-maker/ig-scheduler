📖 怎麼幫業主串接 IG？

  1. 業主的 IG 必須是商業帳號，並且綁定一個 FB 粉專
  2. 請業主把該 IG 的 FB 粉專，把你（聖研）的 FB Developer App 加為合作 App，或把業主帳號加為你 App 的 Tester
  3. 到 Graph API Explorer，切到該業主帳號，勾權限： instagram_basic, instagram_content_publish, pages_show_list, pages_read_engagement, business_management
  4. 產生 Access Token，再用 Token Debugger 換成 60 天長效
  5. 把 token 貼到上面對應業主的「設定 Token」按鈕，工具會自動驗證並抓 IG 商業帳號 ID
  6. 之後 token 由系統每天凌晨 4 點自動續期，60 天滾動
