＋ 新增業主

📖 怎麼幫業主串接 IG？

  1. 業主的 IG 必須是商業 / 創作者帳號，並綁定一個 FB 粉專
  2. 到 Graph API Explorer 切到該業主的 App，勾權限： instagram_basic, instagram_content_publish, pages_show_list, pages_read_engagement, business_management
  3. 產生 Access Token，再用 Token Debugger 換成 60 天長效
  4. 把 token 貼到上面業主的「設定 Token」按鈕，工具會自動驗證並抓 IG 帳號
  5. 之後 token 由系統每天凌晨 4 點自動續期
